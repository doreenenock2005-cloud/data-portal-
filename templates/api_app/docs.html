Open Data API

Programmatic access to Dar es Salaam city datasets — free, open, and no authentication required.

No Authentication Required Freely Accessible RESTful JSON API v1.0

Getting Started

Developers do not need an API key or registration. Make standard HTTP GET requests to access datasets.

https://data.dcc.go.tz/api/v1
Format
JSON
Auth
None Required
Version
v1.0 Stable
API Endpoints
GET/api/v1/datasets

Retrieve all published datasets with filtering and pagination support.

Query Parameters
ParameterTypeRequiredDescription
pageintegerNoPage number (default: 1)
per_pageintegerNoResults per page (default: 20, max: 100)
categorystringNoFilter by category slug (e.g. traffic, population)
searchstringNoSearch in title and description

Example Response:

{ "count": 170, "next": "http://localhost:8080/api/v1/datasets/?page=2", "results": [ { "id": 1, "title": "Traffic Data 2026", "slug": "traffic-data-2026", "description": "Traffic data across all five districts...", "category": { "id": 1, "name": "Traffic Data", "slug": "traffic-data" }, "source_department": "Transport Department", "data_owner": "Dar es Salaam City Council", "region": "Dar es Salaam", "temporal_extent_start": "2026-01-01", "temporal_extent_end": "2026-12-31", "language": "English", "format": "csv", "file_size": "14.2 MB", "records": 45, "download_count": 2340, "created_at": "2026-01-15" } ] }
GET/api/v1/datasets/{id}

Retrieve a single published dataset by its unique ID.

Example Request:

GET /api/v1/datasets/1/

Example Response:

{ "id": 1, "title": "Traffic Data 2026", "slug": "traffic-data-2026", "category": { "name": "Traffic Data" }, "source_department": "Transport Department", "data_owner": "Dar es Salaam City Council", ... }
GET/api/v1/categories

Retrieve all dataset categories with dataset counts.

Example Response:

[ { "id": 1, "name": "Traffic Data", "slug": "traffic-data", "dataset_count": 45 }, { "id": 2, "name": "Population Data", "slug": "population-data", "dataset_count": 30 }, ... ]
GET/api/v1/categories/{slug}/datasets

Retrieve all datasets belonging to a specific category.

Example Request:

GET /api/v1/categories/traffic-transportation/datasets/

Example Response:

{ "count": 27, "results": [ { "id": 1, "title": "Road Traffic Volume 1 2026", "subcategory": { "name": "Road Traffic Volume" }, "category": { "name": "Traffic & Transportation" }, "format": "csv", "records": 8, ... } ] }
GET/api/v1/subcategories/{slug}/datasets

Retrieve all datasets for a specific subcategory within a category.

Example Request:

GET /api/v1/categories/traffic-transportation/subcategories/road-traffic-volume/datasets/
GET/api/v1/subcategories

Retrieve all subcategories across all categories with dataset counts.

Example Response:

[ { "id": 1, "name": "Road Traffic Volume", "category": "Traffic & Transportation", "dataset_count": 9 }, { "id": 2, "name": "Traffic Congestion Levels", "category": "Traffic & Transportation", "dataset_count": 9 }, ... ]
